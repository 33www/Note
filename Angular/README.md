## 目录

* e2e 这是测试用的

* node_modules 这是node环境

* README.md 这是自述文件

* src 这是放要写的代码  

	**app/ 存放代码**  
     
	app.component.ts 组件  
	app.component.html 组件html代码  
	app.component.css 组件css代码  
	app.module.ts 模块  
     
	**assets/ 存放图片**   
	 
	**index.html 主页**  
	
	**style.css 全局样式**  
	
	**favicon.ico 图标**
